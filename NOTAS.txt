ALTERNATIVA A LA VERSIÒN DE VB 6.0 QUE PUEDE DAR FALSOS POSITIVOS Y EN ALGUNAS PCS NO ANDA.
____________________________

Pasos:
1) Instalar la ùltima versiòn de Python
2) Tipear el siguiente comando para instalar los requerimientos necesarios para correr el script:

pip install requests


Pasos para convertir script de python a .EXE:
1) Tipear y instalar esto:
pip install pyinstaller

2) Crea el ejecutable con esto:
pyinstaller update.spec


3) Asegurate de tener el archivo .SPEC bien configurado.
